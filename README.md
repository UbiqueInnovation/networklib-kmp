## Deployment

Each release on Github will be deployed to Ubique's internal Artifactory.

Use the `VERSION_NAME` as defined in the `gradle.properties` file as the release tag, prefixed with a `v`.

* Group: `ch.ubique.kmp`
* Artifact: `network`
* Version: `major.minor.revision`
